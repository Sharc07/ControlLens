## Overview

ControlLens is a full-stack data auditing platform designed to help users upload datasets, automatically detect quality issues, and visualize audit findings through an interactive dashboard.

The platform simulates an enterprise data governance workflow by combining:

- Dataset ingestion
- Automated validation checks
- Data quality scoring
- Finding generation
- Interactive analytics dashboard

---

# Features

## Dataset Management

- Upload CSV datasets
- Store dataset metadata
- Track audit history
- View dataset summaries


## Automated Data Auditing

ControlLens analyzes datasets for:

- Missing values
- Duplicate records
- Invalid formats
- Data inconsistencies
- Quality score calculation


## Interactive Dashboard

Provides:

- Audit summary metrics
- Dataset health scores
- Finding severity breakdown
- Data quality insights


## Authentication

Implemented:

- User registration
- Secure login
- JWT authentication
- Protected API routes
